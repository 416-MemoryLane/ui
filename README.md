# Memory Lane UI

## Requirements
- `node >= 18.4` 
  - https://nodejs.org/en
- `yarn >= 1.22`
  - https://yarnpkg.com/getting-started/install

## Steps to run
- Clone repo
- Install dependencies in both `ui` and `server` directories
  - `cd server && yarn`
  - `cd ui && yarn`
- Run both `ui` and `server` services in separate terminal windows
  - `cd server && yarn start`
  - `cd ui && yarn dev`
